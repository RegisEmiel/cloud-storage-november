Поддерживаются следующие команды
ls - выводит содержимое директории
cd имя_папки - делает текущей указанную папку
cd.. - перейти на уровень выше
cat - вывести текстовый файл на эран
mkdir имя_папки - создать папку с указанным именем
exit - выход

Пробелы в названии директорий и файлов не допускаются.
